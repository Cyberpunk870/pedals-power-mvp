## Short description

Pedals Power is a mobile-first challenge app for registering, uploading proof, and generating posters and certificates.

## Full description

Pedals Power helps brands, communities, and event teams run movement challenges in a simple mobile-first flow.

Participants can register for a challenge, upload activity proof, and unlock social-ready assets without needing a complex app setup.

The current MVP supports:

- challenge registration
- Shopify-based registration intake
- activity proof submission
- participant profile capture
- poster generation
- certificate generation

Pedals Power is built for one-day challenge journeys across cycling, running, walking, swimming, yoga, and similar activities.
